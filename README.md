# 🚀 NOVIUM v0.1 – Ton assistant futuriste



**NOVIUM** est ton **assistant personnel futuriste** qui organise ta vie, analyse ton calendrier, tes notes et les newsletters d’entreprises pour te donner des **insights clairs et rapides**.

---

## 🌌 Fonctionnalités principales

- **Calendrier** : analyse tes événements importants  
- **Notes** : suit tes objectifs personnels et professionnels  
- **Emails d’entreprise** : analyse les newsletters sans jamais espionner tes mails privés  
- **Insight futuriste** : résumé clair et stylé directement dans l’interface  
- **Interface moderne** : design futuriste, intuitif et responsive

## 🛠 Installation & Déploiement

### Option 1 : Déploiement gratuit sur Vercel
1. Crée un compte gratuit sur [Vercel](https://vercel.com/)  
2. Clique **Import Project → From GitHub**  
3. Sélectionne ton repo NOVIUM  
4. Vercel détecte automatiquement Node.js + Express  
5. Ton site est live immédiatement 🚀  

### Option 2 : Tester localement (Codespaces / Replit)
```bash
git clone <ton-repo-github>
